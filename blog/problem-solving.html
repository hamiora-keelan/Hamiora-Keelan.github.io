Elegantly solved a problem.

what was the problem?

The problem was understanding the logic behind a piece of code involving if conditions and the modulo operator (%) when trying to solve the FizzBuzz problem.

what problem-solving techniques did you use?

I first broke the problem down in my head, similar to how you would write pseudocode. This led me to the understanding that an if statement would be used. I then researched using LLM's and google search which led me to understanding how the modulo operator works. I then needed to figure out how the modulo operator would work with the if statement which required trial and error whilst using the error codes from the test.js files.

how did you feel throughout the process?

I felt like I was in a state of flow the entire time, it felt seamless, enjoyable and elegant.

what did you learn?

I learnt that having a methodical approach to problem-solving can have a deep impact on the elegance of its timely completion.
