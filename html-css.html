Positioning explained: Static, Relative, Absolute and Fixed

Introduction

Without positioning, you might as well be living your life in 2D. Mastering the positioning properties will give you the freedom to move content at will. Most programs have a default setting, this also applies to HTML and CSS. The default property for CSS positioning is Static. The Static property positions elements in the same order that they were written in HTML. However, if you don't change the positioning property from static to one of the methods discussed in this article the following properties will not change the position of the content:

  • Top
  • Right
  • Bottom
  • Left

Relative Positioning: The bully

Relative positioning moves an element relative to its original position. It stays within its parent element but removes the element from the document flow. Relative positioning, like a bully, may result in the newly positioned element covering another element if it's placed over it, as relative positioning won't cause other elements to move.

Fixed positioning: The teachers pet

Fixed positioning, positions the element relative to the viewport and ignores any positioning rules applied to the parent element. Fixed elements stay in place following your viewport as you scroll your page, the original position of the element does not leave a blank space.

The giant basketball following you down the page is made possible using position: fixed;

Absolute positioning: The free spirit

Absolute positioning, positions an element relative to its nearest positioned ancestor. If no ancestors have been positioned, being a free spirit the element will position itself relative to the initial containing block (typically the body element), behaving as if it were fixed.

Conclusion

The CSS position property determines element placement. Static positioning orders elements obediently. Relative positioning, like a "bully," adjusts an element within its parent, potentially causing overflow. Absolute positioning acts as a "teachers pet," situating relative to the nearest ancestor or initial block. Fixed positioning, resembling a "free spirit," anchors an element to the viewport, remaining fixed while scrolling.
